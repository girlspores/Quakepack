# Quakepack
A faithful rewrite of Hypixel's minigame Quakecraft in .mcfunction

## FEATURES
- Random spawnpoints on each death
- Dash w/ 2.0s cooldown
- Historical kills and wins tracking
- Lifetime kills leaderboard in lobby
- All currently rotating and historical maps from [Hypixel](hypixel.net/quakecraft)
- 6 new maps, made by [your's truly](twitter.com/cadmiumcolored)  (5 successors (i.e Apex IV), 1 entirely new theme)
